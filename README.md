# client-and-server



\# Broadcast Client-Server



A Python-based client-server application that enables broadcast messaging with SSL/TLS encryption.



\## Features

\- Secure communication using SSL/TLS

\- Broadcast messages to multiple clients

\- Real-time client-server communication

\- Built with Python socket programming



\## Requirements

\- Python 3.x

\- SSL certificates (cert.pem, key.pem)



\## Installation

1\. Clone the repository:

```bash

git clone https://github.com/bhoomikabasavaraj9-cmyk/client-and-server.git

cd client-and-server/broadcast\_client\_server

```



2\. Ensure you have the SSL certificates in the directory



\## Usage



\### Start the Server

```bash

python server.py

```



\### Start the Client

```bash

python client.py

```



\## Project Structure

```

broadcast\_client\_server/

├── server.py          # Server implementation

├── client.py          # Client implementation

├── cert.pem          # SSL certificate

├── key.pem           # Private key

└── README.md         # Documentation
```
